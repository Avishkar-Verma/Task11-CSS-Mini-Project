# Avishkar Verma - Task(11): CSS - Mini Project - Thala 7 Sneaker Store

# How to Run locally:
    1> Clone repository
    2> Open index.html in browser -> this will open the landing/Home page of Thala 7 Sneaker Store.
    3> Click on 'Shop Now' button -> this will take you to the Product Display page.
    4> Click on 'ADD TO CART' button -> this will take you to the Shopping Cart page.
    5> Click on 'Pay Now' button -> this will take you to the Checkout/Payment Page.

# Task to cover :
    Design and implement a web application for a Sneaker Selling Store. Which have a landing page, Product Display Page, Shopping Cart View page And a Payment page.
    Note : Don't use bootstrap, javascript and react. Implement only layout using CSS.

    Add-on Self Initiative : To show web application interactive, I have tried to implement traversal of the Web application by adding reference link of the pages on click event of buttons across pages, Home navbar across pages, and Continue Shopping at the Shopping Cart page. I have also tried to implement some transition on hover and active transformation effect.
